# 智能用药问诊助手（MedGuide）

一个基于**问诊式多轮对话**的家庭常见病用药参考系统。像医生问诊一样，通过逐步提问确定症状、伴随情况与用药禁忌，最终给出分级用药建议、配伍禁忌提醒和就医预警。

> 本系统仅作为非处方药（OTC）的用药知识参考，不能替代执业医师或药师的专业诊断。病情严重或持续不缓解请及时就医。

## 核心特性

- **问诊式对话**：分步追问（部位 → 主症状 → 伴随症状 → 病程 → 用药安全画像），不需要用户自己描述一长串。
- **规则推理引擎**：根据采集到的回答匹配病症，按"首选 / 次选 / 慎用"对候选药物进行分级评分。
- **用药安全核查**：自动结合过敏史、是否孕期/哺乳、年龄段、慢性病、正在服用的药物，对推荐结果做禁忌过滤与高亮警告。
- **配伍与相互作用检查**：内置药物-药物、药物-食物、重复成分检测（如双硫仑反应、对乙酰氨基酚重复超量等）。
- **就医预警（红旗症状）**：识别需要立即就医的危险信号并优先提示。
- **药物百科**：可检索的药品库，查看任意药物的适应症、用法、禁忌与注意事项。
- **用药安全自查**：独立的相互作用检查器，勾选多种在服药物 + 饮食因素，检查配伍禁忌与重复成分。
- **我的用药记录**：问诊方案可一键保存（本地），随时回看；结果支持一键复制导出。
- **纯前端 + 离线 PWA**：无需后端，可安装到手机桌面离线使用。

## 问诊流程

1. **选择不适部位/类别**：呼吸道、消化道、疼痛、皮肤过敏、睡眠情绪、发热等。
2. **确认主症状**：在该类别下选择最符合的主诉。
3. **追问伴随症状**：多选，用于区分病因（如风寒 / 风热感冒）。
4. **询问病程与程度**：持续时间、严重度，触发红旗症状预警。
5. **采集安全画像**：过敏史、孕期/哺乳、年龄段、慢性病、当前在服药物。
6. **生成方案**：分级药物推荐 + 安全警告 + 配伍禁忌 + 就医建议。

## 运行方式

**方式一：本地直接打开**
直接用浏览器打开 `index.html` 即可使用（部分浏览器对 `file://` 下的 Service Worker 有限制，PWA 离线能力建议用方式二）。

**方式二：本地静态服务器（推荐，可启用 PWA 安装）**

```bash
# 任选其一
python -m http.server 8080
npx serve .
```

然后访问 `http://localhost:8080/`，引擎测试页在 `http://localhost:8080/tests/test.html`。

**方式三：在线访问**
见上方 [在线预览](#在线预览)。

## 测试

打开 `tests/test.html` 会自动运行推理引擎的单元测试（问诊流程、诊断结论、安全过滤、配伍检查、红旗预警等），页面顶部显示"通过 X / X"。

## 技术栈

- 原生 HTML5 / CSS3 / JavaScript（ES6 模块化，零运行时依赖）
- PWA（Web App Manifest + Service Worker）
- 规则推理引擎（前端纯函数实现，可单元测试）
